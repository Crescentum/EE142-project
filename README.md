# Dataset Download

This project uses three benchmark datasets:

* MNIST
* SVHN
* CelebA

## Download Datasets

Run the following command:

```bash
python download.py
```

The script will automatically download and prepare:

1. MNIST
2. SVHN
3. CelebA

After downloading them, put them in the ./data folder.

## Environment

Recommended Python version:

```bash
Python 3.10+
```

Install dependencies:

```bash
pip install -r requirements.txt
```
